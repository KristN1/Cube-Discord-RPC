# CubeCraft Discord RPC

Custom rich presence for CuebCraft MC server

### Current Version
`0.0.1`

## How to run

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pypresence.

```bash
pip install pypresence
```
Run the file :)
